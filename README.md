📊 Lead Management & Reporting System
A full-stack Lead Management, Dashboard & Reporting system built as part of the ACOLYTE Technologies interview task.
This project allows users to create, manage, filter, analyze, and export leads with a modern dashboard and clean UI.

🚀 Live Demo
     •	🔗 Frontend: (https://task1-eta-nine.vercel.app/) 
     •	🔗 Backend API: 

🧩 Features
    1. Lead Management (CRUD)
        •	Create new leads 
        •	View all leads 
        •	Update existing leads 
        •	Fields: 
            o	Name 
            o	Mobile 
            o	Email 
            o	City 
            o	Service 
            o	Budget 
            o	Status (New · Interested · Converted · Rejected) 
2. Dashboard Analytics
        •	📌 Total leads count 
        •	📊 Status-wise breakdown 
        •	🌍 City-wise distribution 
        •	🛠 Service-wise distribution 
        •	Interactive charts using Recharts 
3. Reporting System
   •	Filter leads by: 
        o	Date range 
        o	City 
        o	Status 
        o	Service 
    •	View filtered results instantly 
    •	Export reports to CSV 
4. Backend APIs
   •	RESTful APIs built with Node.js + Express 
   •	MongoDB database integration 
   •	Clean and scalable architecture 

🛠 Tech Stack
  Frontend
    •	React.js 
    •	Tailwind CSS 
    •	Recharts 
    •	Axios 
Backend
    •	Node.js 
    •	Express.js 
    •	MongoDB + Mongoose 
    •	CORS enabled API
    •	flask
